🌿O QUE É CSS?🌿

🍃🐌🌱

CSS significa Cascading Style Sheets ou "folhas em estilo de cascata".

O CSS é uma ferramenta usada para estilizar páginas HTML.

Enquanto o HTML cria a estrutura do site, o CSS deixa a página organizada e agradável ao olhar.

Com CSS podemos mudar:

  • 🎨cores
  • 🔤fontes
  • 📏tamanhos
  • ✨animações
  • 📦espaçamento
  • 📱organização da p
🍄Sem CSS, os sites seriam muito simples e quase sem design.

🍄Seletores

Os seletores servem para escolher quais elementos do HTML serão estilizados.

eles são a primeira parte do código CSS.

                h1{
                    color: green;
                }
🌱Nesse exemplo:

h1 é o seletor
todos os títulos h1 ficarão verdes

Alguns exemplos de seletores:

  • h1→ títulos
  • p→ parágrafos
  • body→ corpo da página

🪻Propriedades

As propriedades dizem o que está no elemento.

Elas definem características como:

  • 🎨cor
  • 📏tamanho
  • 🔤fonte
  • 🖼️fundo
            p{
                color: brown;
                font-size: 20px;
            }
🌱Nesse exemplo

color muda a cor do texto
font-size muda o tamanho da fonte

🌻Valores

os valores definem como a propriedade ficará.

Cada propriedade precisa de um valor.

                h2{
                    color: orange;
                
                }
🌱Nesse exemplo:

color é a propriedade
orange é o valor

Alguns exemplos de valores:

  • red→ vermelho
  • 30px→ tamanhos
  • center→ alinhamento central

🌸Estrutura do CSS

Todo código CSS segue essa estrutura:

                seletor{
                    propriedade: valor;
                }

exemplo completo:

                body{
                    background:beige;
                }
🌱Nesse exemplo

  • body→ seletor
  • background→ propriedade
  • beige→ valor

🌼conclusão

O CSS é essencial para criar páginas bonitas, modernas e organizadas.

Aprender seletores, propriedades e valores é o primeiro passo para criar sites incríveis.

Quanto mais você praticar, mais fácil ficarà personalizar suas páginas.
