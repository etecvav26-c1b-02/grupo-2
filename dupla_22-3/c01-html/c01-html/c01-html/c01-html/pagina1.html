O que é HTML

O HTML (HyperText Markup Language) é a linguagem de marcação utilizada para criar páginas na internet. Ele permite organizar textos, imagens, links e outros elementos que aparecem em um site.


Quem criou o HTML

O HTML foi criado em 1991 por Tim Berners-Lee, um cientista da computação que trabalhava no CERN (Organização Europeia para Pesquisa Nuclear). Ele desenvolveu o HTML como parte de um projeto maior chamado World Wide Web.

A ideia era permitir que pesquisadores compartilhassem informações de forma simples através de documentos conectados por links.


Importância do HTML

Com o HTML tornou-se possível estruturar páginas na internet e criar os primeiros sites. A linguagem permitiu organizar textos, imagens e links de forma acessível para qualquer pessoa com um navegador web.

Hoje o HTML continua sendo uma das tecnologias fundamentais para o funcionamento da Web.
