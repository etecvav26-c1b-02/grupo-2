O que é o HTML?

O HTML (HyperText Markup Language) não é uma linguagem de programação, mas sim uma linguagem de marcação. Ela é usada para estruturar o conteúdo que você vê nos sites.

Como funciona a marcação?

O HTML funciona através de "tags" (etiquetas) que envolvem o conteúdo. Quase todas as tags possuem uma abertura e um fechamento:

<tag> Conteúdo aqui </tag>

Principais Elementos:

  • <h1> até <h6>: Definem os títulos da página.
  • <p>: Define um parágrafo de texto.
  • <a>: O elemento mais importante, que cria os links (hipertexto).
  • <ul> e <li>: Usados para criar listas como esta.
  • <img>: Usada para inserir imagens (embora nesta página não existam fotos).

A Estrutura Básica de um Documento

Todo site "puro" como este segue uma hierarquia obrigatória:

Tag Função
<!DOCTYPE html> Informa ao navegador que o arquivo é um documento HTML5.
<html> O "pai" de todos os elementos, onde tudo fica guardado.
<head> Contém informações técnicas que não aparecem na tela (como o título).
<body> Onde fica o conteúdo visível: textos, tabelas e links.
