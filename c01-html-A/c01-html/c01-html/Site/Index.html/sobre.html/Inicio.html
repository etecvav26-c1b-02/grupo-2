A Origem da Internet

A internet surgiu da necessidade de compartilhar informações entre centros de pesquisa. O projeto precursor foi a ARPANET, financiado pelo Departamento de Defesa dos EUA durante a Guerra Fria.

O grande marco técnico foi a criação do protocolo TCP/IP, que permitiu a interconexão de redes variadas.
