O que é o HTML?

O HTML (HyperText Markup Language) é a linguagem de marcação padrão para criar páginas na web. Ele não é uma linguagem de programação, mas sim uma forma de estruturar o conteúdo que vemos nos navegadores, como títulos, parágrafos, links e imagens.

Tim Berners-Lee: O Pai da Web

O HTML foi inventado por Tim Berners-Lee em 1989, enquanto ele trabalhava no CERN (Organização Europeia para a Pesquisa Nuclear). Sua intenção era facilitar o compartilhamento de documentos científicos entre pesquisadores de diferentes instituições.

Sir Tim Berners-Lee

Criador do HTML e do primeiro servidor web.

Evolução das Versões do HTML

Versão Ano de Lançamento Principal Diferença
HTML 1.0 1991 Primeira versão com apenas 18 tags.
HTML 2.0 1995 Padronização de formulários e tabelas simples.
HTML 4.01 1999 Suporte a scripts e separação de estilo (CSS).
HTML5 2014 Suporte nativo a áudio, vídeo e geolocalização.
