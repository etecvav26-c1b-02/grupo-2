A Criação da Internet: Linha do Tempo

A internet não foi "inventada" por uma única pessoa em um único dia. Ela é o resultado de uma evolução de protocolos e infraestrutura que começou décadas antes da Web existir.

Marcos Históricos

  • 1961: Leonard Kleinrock publica o primeiro artigo sobre a teoria de troca de pacotes.
  • 1969: A ARPANET entra em operação. A primeira mensagem foi enviada entre a UCLA e o SRI International. A palavra deveria ser "LOGIN", mas o sistema caiu após as duas primeiras letras ("LO").
  • 1974: Vint Cerf e Bob Kahn publicam o "A Protocol for Packet Network Intercommunication", detalhando o design do TCP.
  • 1983: Em 1° de janeiro, a ARPANET muda oficialmente para o protocolo TCP/IP. Este é considerado por muitos o "aniversário oficial" da Internet moderna.
  • 1989: A rede começa a se tornar comercial, com os primeiros provedores de serviço de internet (ISPs) surgindo.

Conceitos Fundamentais

Diferente do rádio ou da TV, a internet foi desenhada para ser uma rede descentralizada. Isso significa que não existe um "botão de desligar" central; se uma parte da rede falhar, os dados podem encontrar outro caminho para chegar ao destino.

Termo Definição Simples
Pacotes Pequenos pedaços de dados em que as mensagens são divididas para viajar pela rede.
Protocolo Um conjunto de regras que permite que os computadores se entendam.
IP O endereço único de cada dispositivo conectado à rede.
